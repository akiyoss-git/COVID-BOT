Для работы бота необходимо установить дополнительные библиотеки:

pip install -r requirements.txt

Запуст бота:
1) Перейти в командной строке в директорию в которой лежит бот 
2) python TestedBot.py

Тестирование желательно проводить на мобильном устройстве из-за того что на десктопных устройствах невозможно передать геолокацию
